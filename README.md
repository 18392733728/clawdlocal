# ClawdLocal

一个轻量级的本地AI代理框架，专为开发者设计。

## 特性
- 🚀 **本地优先** - 所有数据处理都在本地，保护隐私
- ⚡ **高性能** - 用Go编写核心，快速响应
- 🔌 **插件系统** - 支持Go和Python插件
- 🛠️ **开发者友好** - 内置调试工具和监控
- 📦 **模块化** - 按需加载功能模块

## 架构概览
```
clawdlocal/
├── core/          # 核心引擎
├── plugins/       # 插件系统
├── tools/         # 内置工具集
├── config/        # 配置管理
└── examples/      # 使用示例
```

## 快速开始
```bash
go run main.go
```
